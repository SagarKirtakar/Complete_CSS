# ♟️ Chess Board using HTML5 & CSS3

A beginner-friendly project that creates a classic **8×8 Chess Board** using only **HTML** and **CSS**. This project helps in understanding layouts, Flexbox, colors, and Unicode symbols in web development.

---

## 📌 Project Purpose

The purpose of this project is to practice and strengthen fundamental HTML and CSS concepts by designing a standard chess board.

This project demonstrates:

* Structuring web pages using HTML
* Styling elements using CSS
* Using Flexbox for layouts
* Working with colors and alignment
* Displaying chess pieces using Unicode symbols
* Building responsive and organized UI components

---

## 🚀 Features

✅ Standard 8×8 chess board layout
✅ Alternating black and white squares
✅ Unicode chess pieces for both sides
✅ Centered and properly aligned pieces
✅ Clean and beginner-friendly code
✅ Separate HTML and CSS files

---

## 🛠️ Technologies Used
