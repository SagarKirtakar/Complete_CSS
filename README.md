## 📚 Concepts Covered

### 1. CSS Selectors

#### ID Selector

```css
#main-title {
    text-align: center;
}
```

#### Class Selector

```css
.high-light {
    background-color: orange;
}
```

---

### 2. Colors and Typography

* Background colors
* Text colors
* Font family
* Text alignment

Example:

```css
body {
    background-color: #333;
    color: aliceblue;
    font-family: Arial, Helvetica, sans-serif;
}
```

---

### 3. Box Model

Understanding:

* Margin
* Padding
* Border
* Border Radius

Example:

```css
.box {
    margin: 10px;
    padding: 10px;
    border: 5px solid maroon;
    border-radius: 10px;
}
```

---

### 4. Flexbox Layout

Used to align elements horizontally and manage spacing.

Example:

```css
#flex-container {
    display: flex;
    justify-content: center;
    gap: 10px;
}
```
